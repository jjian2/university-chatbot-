# 🎓 대학 학사정보 챗봇 시스템

## 📌 프로젝트 정보
- 기간 : 2025.04 ~ 2025.11  
- 참여 형태 : 6인 팀 프로젝트  
- 기여도 : 80%

## 📖 프로젝트 소개
STT/TTS 기반 음성 입력을 통해 학사정보를 제공하는 AI 챗봇 서비스입니다.  
학생들이 음성 또는 텍스트로 질문하면 AI가 학사 일정, 공지사항, 주요 정보를 실시간으로 응답하도록 설계되었습니다.

## 🧑‍💻 담당 역할
- 프론트엔드 UI 개발 및 사용자 흐름 설계  
- Flask / FastAPI 서버 연동 및 API 통신 구조 설계  
- 음성 입력(STT) 및 음성 출력(TTS) 기능 연동  
- 서비스 안정성 개선 및 오류 처리 로직 구현  

## 📚 Lesson Learned
- 음성 처리 시스템 통합 및 비동기 통신 구조 실무 경험  
- 프론트엔드와 AI 서버 간 API 연동 설계 역량 강화  
- 대규모 팀 협업 환경에서 역할 분담 및 커뮤니케이션 경험  
- 서비스 안정성과 사용자 경험을 고려한 구조 설계 중요성 인식
